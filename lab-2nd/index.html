Want to try different fonts?


I am using Google fonts, which can easily be added to your page (instructions below).

If you want to give it a try, go to this site and choose your favorite font https://fonts.google.com/
  1. Go to https://fonts.google.com/
  2. Once you find the font you like, click on it
  3. While viewing the details of the font, click "+select this font" at the right of the screen
  4. Copy the code which is provided to your page to use that font
  5. Done!

  • Note that you can select more than one font, and Google will provide you a single "link" import for all of them.
  • You can remove previously selected fonts using the panel at the right.
